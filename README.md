# Moonlandr

## About
Simple fronting dashboard

## Prerequisites
NodeJS

## Installation
```sh
npm i
```

## Config
`conf.json`

- `owner`: The owning user or company name
- `host`: The web address where the primary server lives
- `services`: A List of services and their ports

## Run
```sh
node moonlandr
```

## Deploy
```sh
npm run docker-run
```
